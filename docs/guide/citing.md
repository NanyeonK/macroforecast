# Citing

If you use `macroforecast` in a paper, report, or replication package, cite the
software release by version. The repository root includes `CITATION.cff`, which
GitHub and citation managers can read directly.

Plain text citation:

> Nanyeon Kim. `macroforecast`, version 0.9.5, 2026. <https://github.com/NanyeonK/macroforecast>

The project does not currently have a DOI.
