# User Guide

Use the user guide for the current layer-contract interface. The older stage-numbered pages remain linked where useful, but the canonical runtime map is now L0-L8 plus L1.5-L4.5 diagnostics.

## Design

[Layer Contract Design](design.md) explains the current architecture:

- list layers vs. graph layers;
- construction, diagnostic, and consumption categories;
- default-off semantics for L6, L7, and `.5` diagnostics;
- YAML layer keys and versioned sink names;
- L8 export and manifest behavior.
