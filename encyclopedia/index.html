Encyclopedia

Encyclopedia-style browse for every macroforecast schema choice. Each layer, sub-layer, axis, and option value has its own page or anchor; the tree below is generated from the live LayerImplementationSpec registry plus the OPTION_DOCS documentation registry under macroforecast/scaffold/option_docs/.

Looking for the design narrative instead? Use Architecture – that’s where the prose “why is L2 separated from L3” / “how does L7 read L4 sinks” / cross-layer reference explanations live. Encyclopedia pages here are auto-generated lookup for individual option values (description, when to use, when NOT, references, related options); Architecture pages there are hand-written narrative for the design contracts. Both are sourced from the same LayerImplementationSpec registry – encyclopedia is the machine-locked option dictionary, architecture is the human-edited design guide.
